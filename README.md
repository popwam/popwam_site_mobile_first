# POPWAM Mobile-first SEO Website

نسخة خفيفة جدًا لموقع تعريفي من 6 صفحات + 30 Case Studies.

## تعديل المشاريع
عدّل الملف:
`assets/js/projects-data.js`

لكل مشروع يمكنك تعديل:
- title
- category
- summary
- demoUrl
- mainUrl
- challenge
- solution
- result
- stack

## روابط بدون html
كل صفحة داخل folder باسم `index.html`، لذلك الرابط يظهر مثل:
`/about/` وليس `/about.html`.

## Google Search Console
استبدل محتوى:
`google-site-verification.txt`
بكود التحقق الحقيقي.

## Formspree
في صفحة contact غيّر `https://formspree.io/f/your-id` إلى رابط الفورم الحقيقي.
